Readme

1. The perl script 'global_alignment_single_copy_genes.pl' is used for global alignment of single copy genes.
Running:
perl global_alignment_single_copy_genes.pl <24species_Orthogroups.single.txt> <sequences.list.txt> <type: 'cds' or 'pep'> <output>

2. Example:
perl global_alignment_single_copy_genes.pl 24species_Orthogroups.single.txt sequences.list.txt pep alignment.fa

The related data and files are included in 'Example'.
